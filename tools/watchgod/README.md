# watchgod

## Installation

```sh
$ go install github.com/skerkour/golibs/tools/watchgod
# or
$ git clone https://github.com/skerkour/golibs.git && cd golibs/tools/watchgod && go install
```

## How to use

```bash
$ watchgod -build="go build" -command="./myprogram"
```
